Some extra tips and tricks

Creating a table of contetns

If you have an id attribute on a div, you can use it as the href attribute of an anchor tag to take people to that spot in the page.
i.e. <div id="heading-tags"> </div> could be linked to with <a href="#heading-tags"> Heading Tags</a>

Open link in a new tab

You can force an anchor tag to open the link in a new tab by setting the target attribute to _blank
<a href="https://google.ca" target="_blank"> link </a>
